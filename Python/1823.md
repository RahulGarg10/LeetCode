# 1823. Find the Winner of the Circular Game
## Iterative solution
Time Complexity: $O(n)$

Space Complexity: $O(1)$

```python3
class Solution:
    def findTheWinner(self, n: int, k: int) -> int:
        winner = 0
        for i in range(2,n+1):
            winner = (winner + k) % i
        return winner + 1
```
## Recursive solution
Time Complexity: $O(n)$

Space Complexity: $O(n)$

```python3
class Solution:
    def findTheWinner(self, n: int, k: int) -> int:
        def winner(n):
            # base case
            if n == 1: return 0
            # recursive case
            return (winner(n - 1) + k) % n
           
        return winner(n) + 1
```
