# 896. Monotonic Array
Time Complexity: $O(n)$

Space Complexity: $O(1)$
2nd solution is on my LeetCode account.
```python
class Solution:
    def isMonotonic(self, nums: List[int]) -> bool:
        n = len(nums)
        flag_1 = 0
        flag_2 = 0

        for i in range(n-1):
            if nums[i] < nums[i+1]: flag_1 = 1
            if nums[i] > nums[i+1]: flag_2 = 1
            if flag_1 and flag_2 : return False

        return True
```
