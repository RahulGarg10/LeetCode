# 779. K-th Symbol in Grammar
## Bitwise Method
Time Complexity: $O(n)$

Space Complexity: $O(n)$
``` python3
class Solution:
    def kthGrammar(self, n: int, k: int) -> int:
        #Length of string
        length = 2**(n-1)

        # Midpoint of above string
        midpoint = length // 2

        if n == 1: return 0
        
        if k <= midpoint:
            return self.kthGrammar(n-1, k)
        else:
            return int (not self.kthGrammar(n-1, k - midpoint))
```
