# 977. Squares of a Sorted Array
Time Complexity: $O(n)$

Space Complexity: $O(n)$
```python3
class Solution:
    def sortedSquares(self, nums: List[int]) -> List[int]:
        n = len(nums)
        left_pointer, right_pointer = 0, n-1
        res =[0]*n

        for index in reversed(range(n)):
            if nums[left_pointer]**2 > nums[right_pointer]**2:
                res[index] = nums[left_pointer]**2
                left_pointer += 1

            else:
                res[index] = nums[right_pointer]**2
                right_pointer -= 1

        return res

        
```
